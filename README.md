# Tidy Desktop

Are you tired of constanly having a desktop full of files, or just straight up lazy to drag some files across (like me) this is the perfect python based script tool for you. Just run the code, and it will automatically create a folder named 'Everything' and stuff every file on your desktop into it, quick temporary solution to organize mass clutter of files very quickly. 

# Dependencies / Issues
• Python needs to be installed.


• If One Drive is set up to sync the desktop, it may not work.


• If the file is open it will skip over the said file and move everything else.


• If there is already a folder named 'Everything' in desktop, this script will fail. Working on a fix with automatic file name update.
